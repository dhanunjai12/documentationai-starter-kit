## Quick Setup Guide

Follow these steps to start using Goom Social effectively:

<Steps>
  <Step title="Install the Extension" icon="download">
    Visit the Chrome Web Store and install Goom Social.
  </Step>
  <Step title="Configure Preferences" icon="settings">
    Open the extension settings and customize your preferences.
  </Step>
  <Step title="Start Chatting" icon="message-circle">
    Click the Goom Social icon to begin chatting with the AI.
  </Step>
</Steps>

<Callout kind="tip">Make sure to enable notifications for real-time updates.</Callout>
